# Typst Slides Improvements Summary

## Changes Made

### 1. Shared Macros & Symbols (`shared/macros.typ`)

Created a centralized file for reusable macros and symbols to maintain consistency across presentations and papers.

#### Text Styling Macros
- `#emph-color(body)` - Emphasize text in primary color (red)
- `#textit(body)` - Italic text
- `#textbf(body)` - Bold text
- `#textbf-it(body)` - Bold italic text
- `#emph-bold(body)` - Colored bold text
- `#emph-it(body)` - Colored italic text
- `#code-inline(body)` - Inline code/monospace

#### Common Symbols & Abbreviations
- `#RRI` - Relative Reconstruction Improvement
- `#CR` - Coverage Ratio
- `#CD` - Chamfer Distance
- `#NBV` - Next-Best-View
- `#FiveDoF` / `#SixDoF` - Degrees of Freedom
- `#ASE`, `#EFM3D`, `#EVL`, `#ATEK` - Dataset/model names
- And more...

#### Mathematical Expressions
- `#rri-formula` - Pre-defined RRI formula
- `#coverage-ratio-formula` - Coverage ratio formula
- `#action-space` - Action space definition

### 2. Enhanced Slide Styling

#### Better Use of Template Features
- **`#color-block`**: Used extensively for organizing content with icons
  - Added icons like `"database"`, `"target"`, `"brain"`, `"bulb"`, etc.
  - Color-coded blocks for visual hierarchy

- **`#quote-block`**: Used for highlighting key insights and challenges
  - Custom colors for different contexts

- **Grid Layouts**: Improved two-column layouts for better content organization

#### Typography Improvements
- Bold mathematical vectors using `$bold(q)$` notation
- Consistent use of emphasis macros instead of raw markdown
- Better inline code formatting with `#code-inline`

### 3. GenNBV Integration

Added comprehensive GenNBV slide covering:
- **Key Innovations**:
  - 5DoF free-space action space
  - Multi-source state embedding (geometric, semantic, action)
  - Probabilistic 3D occupancy grid
  - Cross-dataset generalization results

- **Action Space Design**: Mathematical formulation
- **State Representation**: Detailed breakdown of embeddings
- **RL Framework**: PPO-based training with coverage ratio rewards

### 4. Updated VIN-NBV Slide

- Reorganized with color blocks for better structure
- Added mathematical notation with bold vectors
- Included interpretative quote block
- Better visual hierarchy

### 5. Synthesis Slide

Created "Synthesis: RRI-based NBV for Scene-Level Reconstruction" combining:
- Insights from both VIN-NBV and GenNBV
- Mesh-based RRI computation formulas
- Entity-wise RRI aggregation
- Proposed pipeline architecture

### 6. Updated TODOs

Reorganized with color blocks and icons:
- Literature review tasks
- Technical exploration objectives
- Implementation goals

All tasks now reference specific concepts using shared symbols.
