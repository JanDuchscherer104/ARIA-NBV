1 Shared Macros Quick Reference

Import the shared macros in any Typst file:

#import "../shared/macros.typ": *

1.1 Text Styling

Macro Usage Result
#emph-color[text] Primary color emphasis text
#textit[text] Italic text
#textbf[text] Bold text
#textbf-it[text] Bold italic text
#emph-bold[text] Colored bold text
#emph-it[text] Colored italic text
#code-inline[.ply] Inline code .ply

1.2 Common Symbols

Use these symbols to ensure consistency:

#RRI           // Relative Reconstruction Improvement
#CR            // Coverage Ratio
#CD            // Chamfer Distance
#NBV           // Next-Best-View
#GT            // Ground Truth
#FiveDoF       // 5 Degrees of Freedom
#SixDoF        // 6 Degrees of Freedom
#AUC           // Area Under Curve
#PC            // Point Cloud
#SLAM          // Simultaneous Localization and Mapping

// Datasets & Models
#ASE           // Aria Synthetic Environments
#EFM3D         // Egocentric Foundation Model 3D
#EVL           // Egocentric Voxel Lifting
#ADT           // Aria Digital Twin
#AEO           // Aria Everyday Objects
#SSL           // Scene Script Structured Language

1.3 Pre-defined Math Formulas

// RRI formula
#rri-formula
// Expands to:
// $ RRI(q) = (CD(cal(R)_"base", cal(R)_"GT") - CD(cal(R)_("base" union q), cal(R)_"GT")) / (CD(cal(R)_"base", cal(R)_"GT")) $

// Coverage ratio
#coverage-ratio-formula
// Expands to:
// $ CR_t = (tilde(N)_t) / (N^*) dot 100% $

// Action space
#action-space
// Expands to:
// $ cal(A) = bb(R)^3 times S O(2) $

1.4 Usage Examples

1.4.1 Basic Text Styling

The #emph-color[key innovation] is using #RRI for #NBV selection.
We store meshes as #code-inline[.ply] files.
This method uses #FiveDoF action space with #textit[free-space] exploration.

1.4.2 Mathematical Expressions

// Use bold for vectors
$ hat(RRI)(bold(q)) = f(bold(s)_t) $

// Reference predefined formulas
The #RRI is computed as:
#rri-formula

// Combine with other notation
$ bold(s)_t = "Linear"(bold(s)_t^G; bold(s)_t^S; bold(s)_t^A) $

1.4.3 In Color Blocks

#color-block(title: [Key Findings], icon: "bulb")[
  - #emph-bold[#RRI] outperforms coverage-based metrics
  - #ASE dataset enables #emph-it[cross-domain] generalization
  - #EVL achieves #FiveDoF pose estimation
]

1.4.4 In Quote Blocks

#quote-block(color: rgb("#285f82"))[
  The #emph-it[multi-source state embedding] from GenNBV enables better
  #NBV prediction using #CD between partial and #GT meshes.
]

1.5 Adding New Macros

To add new macros, edit shared/macros.typ:

// Add new text style
#let textsf(body) = text(font: "sans-serif", body)

// Add new symbol
#let TSDF = "TSDF"  // Truncated Signed Distance Field

// Add new formula
#let chamfer-distance = $
  CD(S_1, S_2) = sum_(x in S_1) min_(y in S_2) ||x - y||^2
$

1.6 Best Practices

  1. Always use symbols for abbreviations instead of typing them out
  2. Use #emph-color for emphasis instead of raw *asterisks*
  3. Bold vectors in math using $bold(v)$ notation
  4. Prefer predefined formulas for commonly used equations
  5. Group related content in #color-block with appropriate icons
  6. Highlight key insights in #quote-block with custom colors

1.7 Template Integration

The macros work seamlessly with the definitely-not-isec-slides template:

#import "@preview/definitely-not-isec-slides:1.0.1": *
#import "../shared/macros.typ": *

#show: definitely-not-isec-theme.with(...)

#slide(title: [My Slide])[
  #color-block(title: [Results], icon: "chart")[
    Our method achieves #emph-color[98.26% coverage] on #ASE
    using #RRI-based #NBV selection with #FiveDoF action space.
  ]
]
