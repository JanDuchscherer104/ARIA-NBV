# Typst Essentials (Minimal)

## Core Directives
- `#import`, `#include`, `#let` for modularization and variables.
- `#set`, `#show` for styling and show rules.

## Labels & References
- Attach `<label>` to figures/tables/sections.
- Reference with `@label`.
- Citations use `@bib_key` (or `#cite(...)` if needed).

## Layout Primitives
- `#grid(...)`, `#stack(...)`, `#h(...)`, `#v(...)`.

## Figures & Tables
- Wrap images/tables in `#figure(...)` with `caption:` and `<label>`.
