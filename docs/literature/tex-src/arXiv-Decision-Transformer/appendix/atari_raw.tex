
\section{Atari Task Scores} \label{appendix:atari_raw}

Tables \ref{tbl:atari_main_raw} and \ref{tbl:atari_percentile_bc_raw} show the raw scores corresponding to Tables \ref{tbl:atari_main} and \ref{tbl:atari_percentile_bc}, respectively. For \%BC scores, we use the same hyperparameters as Decision Transformer for fair comparison.  For REM and QR-DQN, there is a slight discrepancy between \citet{agarwal2020optimistic} and \citet{kumar2020conservative}; we report raw data provided to us by REM authors.
