python -m oracle_rri.data.downloader -m download -c efm_eval -n 5 --ms 10

This will download the top 5 scenes with the most snippets, using the default configuration (here: EFM training set).

python -m oracle_rri.data.downloader -m download -c efm -n 5

2 Dataset config & auto-discovery

The ASEDatasetConfig provides several options for configuring the dataset loading process. See the docstring for details.

  • Auto tar resolution: if tar_urls is empty, shards are globbed under resolved ATEK root; if scene_ids are provided, only those subdirs are used.
  • Mesh resolution: when load_meshes=True, meshes are loaded from .data/ase_meshes/scene_ply_{scene}.ply (or inferred from scene_ids).

# Dataset config & auto-discovery

The `ASEDatasetConfig` provides several options for configuring the dataset loading process. See the docstring for details.

- **Auto tar resolution**: if `tar_urls` is empty, shards are globbed under resolved ATEK root; if `scene_ids` are provided, only those subdirs are used.
- **Mesh resolution**: when `load_meshes=True`, meshes are loaded from `.data/ase_meshes/scene_ply_{scene}.ply` (or inferred from `scene_ids`).


```{python}
from oracle_rri.data.dataset import ASEDatasetConfig


cfg = ASEDatasetConfig(verbose=True)
cfg.inspect()

ds = cfg.setup_target()
```

The dataset object `ds` yields samples of type "TyppedASESample", this dataclass stores the complex flat-dict from the `AtekWDSDataset` in `ds.flat` and provides typed views via properties.

```{python}
sample = next(iter(ds))
sample.rich_summary(show_only_sample=["efm_gt"])
```
